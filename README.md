# DevRoast 🔥

Your code, brutally roasted.

DevRoast is a platform where developers submit their code and receive brutally honest feedback. Think of it as a code review from that one coworker who's never afraid to tell you the truth.

## Features

- **Paste & Roast** - Submit your code and get instant feedback
- **Score System** - See how bad (or good) your code really is
- **Leaderboard** - Compare your roasts with other developers
- **Detailed Analysis** - Get line-by-line feedback on what went wrong
